# Website Project "Bittner"

## Description

A website created during an internship at [Czek.it](https://czek.it/).  

Preview: [Bittner](https://mikoslaf.github.io/bittner/)

The Bittner project is a responsive website with a mobile version based on a graphic design created in Adobe XD. 
I developed the website during an internship in 2021, when my programming skills were less advanced.

### Libraries/Technologies Used:
* [Gulp 2.3.0](https://gulpjs.com/docs/en/getting-started/quick-start/)
* jQuery 3.6.0
* [SCSS 1.42.1](https://sass-lang.com/)
* [Owl Carousel](https://owlcarousel2.github.io/OwlCarousel2/)

## Installation

To run the project locally, simply clone the repository using the command:
```bash
git clone https://github.com/mikoslaf/bittner.git
```
